## Project Overview

This is an **Altium Designer (v23.3.1)** hardware project for a brushless motor (BLDC) control board based on the **STSPIN32G4** — an STMicroelectronics system-in-package that integrates a Cortex-M4 MCU with a 3-phase gate driver. The project is intended for learning and controlling brushless DC motors.

## Key Files

- **[MotoCtrl_Rev_1.0.0.PrjPcb](Hardware/MotoCtrl_Rev_1.0.0/MotoCtrl_Rev_1.0.0.PrjPcb)**: The Altium project file. Defines all schematic sheets, PCB, libraries, ERC rules, output job configurations, and annotation settings.
- **[STSPIN32G4-Test-board.SchLib](Hardware/MotoCtrl_Rev_1.0.0/STSPIN32G4-Test-board.SchLib)**: Schematic symbol library, contains the custom STSPIN32G4 component symbol.
- **[STSPIN32G4-Test-board.PcbLib](Hardware/MotoCtrl_Rev_1.0.0/STSPIN32G4-Test-board.PcbLib)**: PCB footprint library for custom footprints.

## Schematic Hierarchy

The design is organized as a flat multi-sheet schematic:

| Sheet | Purpose |
|---|---|
| `TopLevel.SchDoc` | Top-level sheet with inter-sheet connections |
| `MCU.SchDoc` | STSPIN32G4 microcontroller, decoupling, programming/debug header |
| `DRV.SchDoc` | 3-phase gate driver outputs, current sensing, motor connectors |
| `POW.SchDoc` | Power input, voltage regulation (buck/LDO), power filtering |
| `COMM.SchDoc` | Communication interfaces (UART, SPI, CAN, etc.) |
| `HMI_SENSOR.SchDoc` | User interface (buttons, LEDs) and sensor inputs (Hall, encoder, temperature) |

## Altium-specific Notes

- **Project files are binary** (`.SchDoc`, `.PcbDoc`, `.SchLib`, `.PcbLib`) — they cannot be diffed as text. Only Altium Designer can open them.
- **History/** and **__Previews/** folders are gitignored — these are auto-generated by Altium.
- Output paths (Gerber, BOM, PDFs) are configured in the project but the output files themselves are gitignored.
- The project uses **ERC (Electrical Rules Check)** with custom connection matrix settings — verify ERC passes before committing changes.

## Workflow Notes

- All design work must be done in **Altium Designer v23.3.1** or compatible.
- Before committing schematic/PCB changes, run **ERC** and **DRC** (Design Rules Check) to verify no violations.
- The `.gitignore` already excludes Altium temporary files, output files (Gerber, BOM, PDF), and OS files.
- Schematic previews in `__Previews/` are auto-generated and gitignored; `History/` zip backups are also gitignored.
