## 功能模块说明

| 模块 | 原理图 | 功能 |
|---|---|---|
| **电源 (POW)** | `POW.SchDoc` | 输入电源滤波、DC-DC 降压、3.3V LDO、12V 栅极驱动供电 |
| **主控 (MCU)** | `MCU.SchDoc` | STSPIN32G4 最小系统，SWD 调试接口，去耦电容，BOOT 配置 |
| **驱动 (DRV)** | `DRV.SchDoc` | 三相 MOSFET 逆变桥，电流采样，Hall/编码器接口，电机连接器 |
| **通信 (COMM)** | `COMM.SchDoc` | UART/SPI/CAN/USB 接口，外部通信连接器 |
| **人机交互 (HMI)** | `HMI_SENSOR.SchDoc` | LED 指示灯，按键，电位器调速 |
